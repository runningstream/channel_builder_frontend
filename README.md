# Description

This is the Vue 3 / Vite frontend to runningstream.cc

# Running

Deploy to S3 and cloudfront

# Development

Setup - `npm install`

Dev loop - `npm run dev -- --host`

Typescript checker loop - `npx vue-tsc --watch --noEmit`

# Building

`npm run build`
